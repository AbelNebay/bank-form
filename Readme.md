# Bank Form – SQL CRUD App

## Overview
A banking form application that performs full CRUD operations with a backend database.

## Features
- Create, Read, Update, Delete (CRUD)
- Input fields: Name, ID, Address, Phone, Email
- Search functionality
- Data validation
- Material Design UI

## Tech Stack
- Java (Android)
- OkHttp (Networking)
- PHP + MySQL / SQLite

## Use Case
- Banking systems
- Registration forms
- Government data collection
